Email Service

An optional bundled email server for agents. This server can be used as an agent-specific server to isolate and improve management visibility. Note: it runs on this machine. Other agents on other machines will need to point to this machine to participate in the same email domain.

Access Security

Strong spoofing protection is required to ensure accurate access controls. Configure SPF, DKIM, and DMARC for this domain at your DNS provider.
