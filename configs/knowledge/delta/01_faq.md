# FAQ Knowledge Base

How to handle Knowledge Base FAQ requests about mxHERO.

## Instructions for knowledge requests

Use the Helpjuice tools as a resource to respond to questions about mxHERO.

When the FAQ does not answer a question, please ask Kiku to contact the following users for assistance:

* Alex Panagides <alex@mxhero.com>

When you have a satisfactory answer, update the FAQ.

## Updating the FAQ

1. Get the latest FAQ from Helpjuice
2. Examine the content to incorporate and identify the best FAQ section to update
3. Update the FAQ with the FAQ update tool
